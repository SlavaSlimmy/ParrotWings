# ParrotWings

## Install UI
````bash
cd interface
npm i
bower i
````

## Install server
````bash
cd app/backend
npm install
````

## Start server
````bash
cd app/backend
node server.js
````

## Install app
````bash
cd app
npm install
npm run clean & npm run build
````

## Start app

First need to start server, then app.

````bash
cd app
npm start
````
